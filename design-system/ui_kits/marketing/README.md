# Citelines · Marketing Site UI Kit

The unauthenticated marketing site for Citelines.

> **Source note.** No real marketing site or Figma was attached — these screens
> are a designer pick that exercises the design system on a public-facing
> surface.

## Screens

- `index.html` — single-page marketing site. Nav, hero, features, social
  quote, pricing, CTA band, footer. All static HTML using the tokens in
  `../../colors_and_type.css`.

## How to run

Open `index.html`. No JS framework — plain HTML + CSS so it serves as a
clear specimen of the brand's voice and chrome.
